1. Date객체 생성

- new Date()
- new Date(98765678978) : 1970.1.1일 기준으로 경과한 날짜
- new Date("2010.02.23")
- new Date(2010,01,23)
- 월은 0월부터 시작 , 숫자로 Date 객체 생성시 하나 작은 값으로 설정 한다



2. Date객체 생성

- now = new Date()
- now.toLocaleString() : 날짜와시간
- now.toLocaleDateString() : 날짜
- now.toLocaleTimeString() : 시간



3.요일구하기
